# Como Contribuir

Obrigado pelo interesse em contribuir! Este guia vai te ajudar a adicionar materiais ao repositório.

---

## Formas de Contribuir

- Adicionar resumos de disciplinas
- Compartilhar listas de exercícios
- Incluir provas anteriores
- Adicionar links úteis
- Criar guias de estudo
- Corrigir erros em materiais existentes

---

## Estrutura de uma Disciplina

Cada disciplina segue este padrão:

```
nome-da-disciplina/
├── README.md
├── visao-geral-e-dicas.md
├── dificuldades-comuns.md
├── links-uteis.md
├── extras.md
├── resumos/
├── listas-e-exercicios/
├── provas-e-avaliacoes/
├── projetos/
└── materiais/
```

---

## Como Adicionar uma Disciplina

1. **Fork** este repositório
2. Crie a pasta em `periodos/XX-periodo/nome-da-disciplina/`
3. Adicione os arquivos necessários
4. Commit suas mudanças
5. Abra um Pull Request

---

## Padrões de Commits

Use mensagens descritivas:

- `feat: adiciona resumo de Cálculo 1`
- `fix: corrige link quebrado em Redes`
- `docs: adiciona lista de exercícios de SO`
- `feat: adiciona prova de Banco de Dados`

---

## O que Evitar

- Materiais protegidos por direitos autorais
- Provas completas com gabarito (apenas exercícios modelo)
- Conteúdo ofensivo ou inadequado

---

## Dúvidas?

Abra uma issue para perguntar ou peça ajuda no PR.
