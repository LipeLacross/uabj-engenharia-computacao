# UABJ - Engenharia da Computacao

Estrutura inicial de repositorio para organizar materiais colaborativos do curso de **Engenharia da Computacao da UABJ**.

## Organizacao

O repositorio foi separado por **periodos** e, dentro de cada periodo, por **disciplinas**.

Cada disciplina ja vem com uma estrutura-base:

- `README.md`
- `resumos/`
- `materiais/`
- `listas-e-exercicios/`
- `provas-e-avaliacoes/`
- `projetos/`
- `visao-geral-e-dicas.md`
- `dificuldades-comuns.md`
- `links-uteis.md`
- `extras.md`

## Pastas especiais

- `atividades-complementares/`
- `certificacao-intermediaria/`
- `enade/`
- `guias/`
- `templates/`
- `scripts/`

## Observacao

Esta estrutura foi montada com base no cronograma curricular enviado em imagem.
As optativas foram incluídas como placeholders (`optativa-1`, `optativa-2`, etc.) para você preencher depois com os nomes reais.
